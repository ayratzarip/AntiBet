# AntiBet - Дневник самонаблюдения

Telegram Mini App для пациентов с патологическим влечением к азартным играм, проходящих лечение у психотерапевта.

## Описание

AntiBet - это приложение для ведения дневника самонаблюдения, помогающее пациентам отслеживать момен когда появляются мысли об азартных играх. Приложение позволяет фиксировать:

- **Дату и время** появления мыслей
- **Место**, где возникли мысли об игре
- **Свидетели** - присутствовал ли кто-то рядом (знакомые, посторонние или пациент был один)
- **Обстоятельства** - эмоциональный и физический фон (устал, голоден, поругался с женой и т.д.)
- **Триггер** - провокатор, который привел к появлению мыслей
- **Мысли** - содержание мыслей (желание поиграть, самокритика, стыд из-за долгов)
- **Телесные ощущения** - физические реакции тела
- **Действия** - что пациент делает в этой ситуации (пытается отвлечься, запрещает себе думать и т.д.)

## Технологии

- **React** 19 с TypeScript
- **Tailwind CSS** 4 для стилизации
- **React Router** для навигации
- **Telegram Web App SDK** для интеграции с Telegram
- **Vite** для сборки
- **Telegram Cloud Storage** для хранения данных

## Структура данных

Каждая запись включает:
```typescript
{
  id: string;
  location: string;          // Место
  witnesses: string;         // Свидетели
  circumstances: string;     // Обстоятельства
  trigger: string;           // Триггер
  thoughts: string;          // Мысли
  bodyFeelings: string;      // Телесные ощущения
  actions: string;           // Действия
  emoji: string;             // Автоматически подобранная эмоция
  title: string;             // Автоматически сгенерированный заголовок
  tags: string[];            // Автоматически извлеченные теги
  createdAt: string;         // Дата создания
  updatedAt: string;         // Дата последнего обновления
}
```

## Установка и запуск

### Предварительные требования

- Node.js 18+
- npm или yarn

### Локальная разработка

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Деплой на GitHub Pages
npm run deploy
```

## Деплой

Приложение настроено для автоматического деплоя на GitHub Pages:

1. Соберите проект: `npm run build`
2. Задеплойте: `npm run deploy`

Приложение будет доступно по адресу: `https://[username].github.io/AntiBet/`

## Интеграция с Telegram

### Создание бота

1. Создайте бота через [@BotFather](https://t.me/BotFather)
2. Установите Telegram Web App URL в настройках бота
3. Активируйте Cloud Storage для сохранения данных пользователей

### Настройка

В настройках бота укажите URL вашего приложения:
```
/setmenubutton
/newapp
```

## Функциональность

### Основные возможности

- ✅ Создание записей о моментах появления мыслей об игре
- ✅ Просмотр истории записей с группировкой по датам
- ✅ Редактирование существующих записей
- ✅ Удаление записей
- ✅ Поиск по записям
- ✅ Автоматическое определение эмоций и тегов
- ✅ Экспорт всех записей для анализа с AI
- ✅ Поддержка темной темы
- ✅ Адаптивный дизайн

### Анализ с помощью AI

Приложение позволяет экспортировать все записи в формате, оптимизированном для анализа с помощью AI-ассистентов (ChatGPT, Claude и др.). Это помогает:

- Выявить триггеры и паттерны
- Определить связь между обстоятельствами и желанием играть
- Проанализировать эффективность защитных механизмов
- Оценить динамику изменений
- Получить рекомендации для психотерапии

## Безопасность и конфиденциальность

- Все данные хранятся в Telegram Cloud Storage пользователя
- Приложение не имеет собственного бэкенда
- Данные не передаются третьим лицам
- Доступ к записям имеет только владелец Telegram-аккаунта
